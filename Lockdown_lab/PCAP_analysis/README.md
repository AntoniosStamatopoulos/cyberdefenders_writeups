PCAP Analysis – Lockdown Lab
Overview

This folder contains the network traffic analysis conducted as part of the Lockdown DFIR lab. The analysis focuses on identifying attacker behavior within captured network data and understanding how the intrusion unfolded at the network level.

Contents

The folder includes:

Findings:
Documentation of key observations derived from the PCAP analysis, including suspicious patterns, attacker activity, and notable network events.
Screenshots:
Visual evidence captured during analysis (e.g., Wireshark views), supporting the identified findings and illustrating important traffic patterns.
Analysis Focus

The PCAP analysis covers:

Detection of reconnaissance and scanning activity
Identification of suspicious HTTP requests and potential exploitation attempts
Examination of SMB traffic for signs of lateral movement or file transfer
Identification of outbound connections related to command-and-control communication
Correlation of network activity with other forensic artifacts (memory and malware)
Purpose

The goal of this folder is to provide a clear and structured view of the network-level investigation, supporting the overall reconstruction of the incident.

Notes

All findings are derived from offline analysis of the provided PCAP file and are intended for educational and forensic documentation purposes.
